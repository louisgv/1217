proposal

l'eau.web is an interactive web app made to promote l'eau, an adventure game where the main goal is to build an oasis in the middle of a post-apocalyptic desert.

The web app will have 3 pages: one for the the world, one for the characters, and lastly a credit page. These pages let user explore and learn more about the l'eau universe.

The main goal of the web app is to sell the idea behind l'eau.
